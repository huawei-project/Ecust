# 划分数据集

1. 划分方式与上阶段一致，在每人的数据中，保留`Multi`与`RGB`同时检测出的图片路径，打乱后按一定比例划分；
2. 本次实验划分时不做特殊处理，若需要其中指定条件的数据，可在`RecognizeDataset`中指定参数；
